# 差分統合理論

本ディレクトリは、クオリア、主観、差分認識、感情、思考、学習、意思決定を
統一的に説明する「差分統合理論」に関する論考をまとめたものである。

差分統合理論は、生命、意識、感情、思考、学習、意思決定を
個別の現象としてではなく、単一の原理から説明することを目的とする。

## 理論の前提

本理論は、生命および認知システムは
環境との相互作用の中で差分を検出し、
その差分を内部基準に基づいて評価するシステムである
という前提に立つ。

主観、感情、思考、学習、意思決定は、
この差分評価システムの異なる表現・機能であると考える。

## 理論の基本構造

環境  
→ 差分  
→ 内部基準  
→ 主観  
→ クオリア  
→ 感情  
→ 思考  
→ 学習  
→ 意思決定  
→ 行動  

本ディレクトリ内の文書は、同じ理論を別角度から展開・統合したものであり、
単なる重複ではない。

本理論は複数の論文によって構成されているが、
最初に読む文書として入口論文を用意している。
