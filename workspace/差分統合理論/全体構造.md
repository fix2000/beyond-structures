# 差分統合理論 全体構造

本理論は、差分認識を基盤として、
クオリア、主観、感情、思考、学習、意思決定を
一つの連続した構造として説明する統合理論である。

## 基本原理

生命は環境との差分を検出することで生存する。
差分を検出するためには内部基準が必要となる。
内部基準を基点として差分を表現したものが主観であり、
差分の主観的表示がクオリアである。

## 全体構造

環境  
→ 差分  
→ 内部基準  
→ 主観  
→ クオリア  
→ 感情  
→ 思考  
→ 学習  
→ 意思決定  
→ 行動  
→ 生存
