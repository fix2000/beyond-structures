# Beyond Structures
## 三心思想・制度論・文明の再設計

私は長い時間をかけて、人間の在り方、社会の仕組み、制度の歪みについて考えてきた。  
このサイトは、私が整理し続けてきた思想群を、読みやすい形でまとめたものになる。

現代社会は、倫理・道徳・知性の均衡が崩れ、その歪みが政治や経済にも反映されている。  
私は、これら三つの要素を「三心」と呼び、これを基点として制度や教育の再構築を考えている。

ここに置いた文書は、いずれも単発の論考ではなく、  
全体としてひとつにつながった思想体系の一部になる。

---

# 1. 三心思想（基礎）
三心（倫理・道徳・知性）は、人の判断や行動の基盤になるものだと私は考えている。  
社会の問題の多くは、この三つのずれから生じているように見える。

→ [三心思想](three-minds.md)

---

# 2. 令和版科挙制度
学歴や暗記ではなく、三心に基づいて人の判断力や人格を測る制度として考えている。  
選抜のあり方を変える必要性は強く感じている。

→ [令和版科挙制度](kakyo.md)

---

# 3. 知性院構想
政治や行政の外側に、国民のための第三の視点を置く必要がある。  
私は、知性院を国民のための中立的な提言機関として考えている。

→ [知性院構想](intelligence-agency.md)

---

# 4. 教育の再設計
教育は、人格を形作るための基盤だと私は考えている。  
知識だけではなく、倫理や道徳、ものごとを見る視点を育てる必要がある。

→ [教育について](education.md)

---

# 5. AIと情報環境
AIは便利ではあるが、思考や判断を委ねれば人は弱くなる。  
私は、AI時代に求められる人間の側の姿勢について記している。

→ [AI・情報構造](ai.md)

---

# 6. 経済・労働・貨幣
現場労働の価値が軽視され、資本偏重の社会になっている。  
私は、労働の価値と貨幣の扱いについて、長い視点から考えている。

→ [経済・労働・貨幣](economy.md)

---

# 7. 補助文書・思想メモ
三心思想の背景にある考え方や、日々の対話や思索の中で整理されてきた要素をまとめてある。

→ [補助文書](philosophy.md)

---

# このサイトについて
私は、これらの思想を一度で完成させるつもりはない。  
考え続け、その結果を積み上げていくことで、少しずつ形が固まっていくものだと思っている。
